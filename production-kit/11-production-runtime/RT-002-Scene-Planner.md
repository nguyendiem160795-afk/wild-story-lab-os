# Purpose

Scene Planner transforms a story into a structured sequence of production-ready scenes.

It determines scene boundaries, timing, objectives, transitions, and production requirements while maintaining narrative flow and educational effectiveness.

---

# Objectives

- Break stories into logical scenes
- Define scene objectives
- Balance pacing
- Maintain character continuity
- Optimize for AI video generation
- Support multiple video formats

# Scene Structure

Every scene contains:

- Scene ID
- Scene Name
- Story Purpose
- Learning Objective
- Duration
- Characters
- Environment
- Props
- Camera Profile
- Motion Profile
- Expression Profile
- Lighting Profile
- Audio Profile
- Transition
- Prompt Reference

---

# Scene Types

## Hook Scene

Purpose

Capture attention within the first 3 seconds.

Typical Duration

3–5 seconds

Priority

Critical

---

## Introduction Scene

Purpose

Introduce the character or topic.

Typical Duration

5–10 seconds

---

## Teaching Scene

Purpose

Deliver the core educational content.

Typical Duration

8–15 seconds

---

## Challenge Scene

Purpose

Present a problem or interactive moment.

Typical Duration

5–10 seconds

---

## Resolution Scene

Purpose

Solve the problem and reinforce learning.

Typical Duration

5–10 seconds

---

## Ending Scene

Purpose

Summarize the lesson and encourage engagement.

Typical Duration

5–8 seconds

---

# Scene Planning Rules

Every scene must have:

✓ One clear objective

✓ One dominant action

✓ One primary emotion

✓ One camera focus

✓ One transition in

✓ One transition out

---

# Timing Guidelines

| Video Length | Recommended Scenes |

# Best Practices

- Keep one educational objective per scene.
- Avoid multiple major actions.
- Alternate camera angles between scenes.
- Vary expressions naturally.
- End each scene with a clear transition.
