# Lighting Rules

Always

- Keep subjects clearly visible.
- Match lighting with the story mood.
- Maintain consistency within the same scene.
- Use soft shadows for child-friendly visuals.

Never

- Extremely dark scenes.
- Flashing lights.
- High-contrast horror lighting.
- Inconsistent light direction.

---

# Best Practices

- Use warm lighting for educational content.
- Reserve fantasy lighting for magical moments.
- Keep lighting simple and readable.
- Highlight the main character without distracting the viewer.

---

# Common Mistakes

- Overexposed scenes.
- Underexposed characters.
- Multiple conflicting light sources.
- Random color shifts between shots.
