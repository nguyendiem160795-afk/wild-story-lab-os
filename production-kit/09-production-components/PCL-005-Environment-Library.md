# Environment Rules

Always

- Keep the environment clean and organized.
- Support the story without distracting the audience.
- Include interactive objects relevant to the lesson.
- Use bright, welcoming colors.

Never

- Overcrowd the background.
- Add unrelated props.
- Use frightening scenery.
- Change the environment abruptly within a scene.

---

# Best Practices

- Reuse core environments for brand consistency.
- Add small decorative variations instead of redesigning the entire scene.
- Match the environment with the lesson objective.
- Ensure enough open space for character movement.

---

# Common Mistakes

- Too many background details.
- Poor scale between characters and objects.
- Inconsistent environment across scenes.
- Mismatched props.
