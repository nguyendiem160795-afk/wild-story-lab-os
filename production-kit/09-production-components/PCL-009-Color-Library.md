# Color Harmony

Recommended

- Complementary
- Analogous
- Triadic

Avoid

- Random color combinations
- Oversaturated palettes
- Excessive neon colors

---

# Accessibility Guidelines

Always

- Maintain strong contrast.
- Keep educational objects clearly distinguishable.
- Use colors consistently.

Never

- Convey important information using color alone.
- Use flashing color changes.
- Place similar colors together without contrast.

---

# Best Practices

- Keep the main character visually distinct from the background.
- Use warm colors for educational scenes.
- Reserve magical colors for special moments.
- Repeat brand colors throughout each episode.

---

# Common Mistakes

- Too many bright colors competing for attention.
- Inconsistent palette between scenes.
- Low contrast text and objects.
- Background colors overpowering the main subject.
