# Purpose

Production Components Library (PCL) is the reusable building block system for Wild Story Lab OS.

Instead of writing prompts from scratch, creators assemble prompts by combining standardized production components.

This ensures:

- Consistent visual quality
- Faster prompt creation
- Reusable creative assets
- Better scalability
- AI tool compatibility

---

# Architecture

Production Components are divided into three categories.

## Visual Components

- Camera
- Shot
- Lighting
- Composition
- Color Style
