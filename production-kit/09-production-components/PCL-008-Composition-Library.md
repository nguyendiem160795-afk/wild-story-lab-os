# Composition Selection Guide

| Scene Type | Recommended Composition |
|------------|-------------------------|
| Character Introduction | Center Composition |
| Teaching | Rule of Thirds |
| Reading | Natural Framing |
| Adventure | Foreground + Depth |
| Emotional Moment | Close Framing |
| Group Activity | Balanced Composition |
| Ending Scene | Symmetrical Composition |

---

# Composition Rules

Always

- Keep the main subject clearly visible.
- Leave adequate space around characters.
- Avoid cluttered layouts.
- Ensure background supports the story.

Never

- Cut off important body parts.
- Overfill the frame.
- Distract with unnecessary props.
- Place multiple focal points in one shot.

---

# Best Practices

- Position the main character before adding props.
- Use depth to create immersive scenes.
- Align composition with camera movement.
- Keep children's attention focused on one primary action.

---

# Common Mistakes

- Busy backgrounds.
- Cropped characters.
- Unbalanced framing.
- Poor eye-line direction.
- Competing focal points.
