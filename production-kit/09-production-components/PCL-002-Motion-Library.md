# Purpose

This document defines the standardized character motion language for Wild Story Lab OS.

Every animation should use these motion conventions to ensure natural, expressive, and child-friendly character movement across AI video generation platforms.

Supported Platforms:

- Google Flow
- Veo 3.1
- Imagen Video
- Runway
- Sora

---

# Motion Categories

## Idle Motions

- Standing
- Sitting
- Looking Around
- Breathing
- Waiting
- Thinking

Purpose

Create natural pauses between actions.

---

## Walking Motions

- Slow Walk
- Happy Walk
- Confident Walk

- Sneaking Walk

Best Use

## Running Motions

# Motion Intensity

Low

Small and gentle movements.

Medium

Natural everyday actions.

High

Energetic actions suitable for exciting moments.

---

# Motion Selection Guide

| Scene | Recommended Motion |
|--------|--------------------|
| Greeting | Wave |
| Teaching | Pointing, Writing |
| Reading | Turning Pages |
| Adventure | Exploring |
| Cooking | Stirring, Mixing |
| Celebration | Dancing, Clapping |
| Emotional | Slow Walking |

---

# Motion Rules

Always

- Keep movements smooth.
- Match the character's personality.
- Use child-friendly gestures.
- Maintain consistent body proportions.

Never

- Use violent actions.
- Use unnatural body deformation.
- Overuse exaggerated movement.
- Stack too many actions in one shot.

---

# Common Mistakes

- Multiple unrelated actions in one scene.
- Sudden speed changes.
- Ignoring character emotion.
- Repeating the same animation too often.
