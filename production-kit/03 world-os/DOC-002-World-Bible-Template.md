# World Bible Template

> Official template for documenting and managing worlds inside Wild Story Lab.

---

# 1. Purpose

This document provides the standard template for creating a complete World Bible.

A World Bible is the single source of truth for a world.

It defines:

- World identity.
- World rules.
- Environments.
- Locations.
- Characters.
- Story possibilities.
- Evolution paths.

---

# 2. World Bible Definition

A World Bible is the complete knowledge system of a fictional world.

Structure:

```
World Identity

+

World Rules

+

Environment System

+

Location System

+

Character Relationship

+

Story System

+

Evolution System
```
