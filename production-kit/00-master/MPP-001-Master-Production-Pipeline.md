# Phase 1 – Pre Production

Objective:

Prepare everything before opening Google Flow.

Checklist:

□ Select Playbook

□ Select Story Template

□ Define educational objective

□ Select vocabulary

□ Select characters

□ Select background

□ Prepare assets

Output:

- Storyboard
- Production Brief
- Prompt List

---

# Phase 2 – Story Production

Input:

Story Template

Output:

8-scene storyboard

Each scene includes:

- Objective
- Dialogue
- Action
- Camera
- Emotion

---

# Phase 3 – Prompt Production

Convert storyboard into Flow-ready prompts.

Rules:

- Follow DOC-006
- One scene = One prompt
- One prompt = One line
- Character consistency required

Output:

Prompt Pack

---

# Phase 4 – Google Flow Production

Input:

Prompt Pack

Character Assets

Background Assets

Tasks:

Generate scenes

Review animation

Adjust camera

Regenerate if necessary

Output:

Raw Video

---

# Phase 5 – Voice Production

Generate:

Narration

Character voice

Pronunciation

Dialogue

Output:

Voice Track

---

# Phase 6 – Audio Production

Add:

Background music

Sound effects

Ambient sound

Output:

Final Audio Mix
