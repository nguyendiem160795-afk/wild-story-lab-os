# Scene To Shot System

> Official cinematic scene breakdown and shot design framework for Wild Story Lab Production Operating System.

---

# 1. Purpose

This document defines the official Scene To Shot System.

The purpose is to transform story scenes into structured cinematic shot plans ready for AI video production.

---

# 2. Scene To Shot Definition

Scene To Shot is:

```
The process of converting

a narrative scene

into visual camera sequences

for production.
```

---

# 3. Core Philosophy

Wild Story Lab follows:

```
Every Shot Has A Purpose.

Every Camera Movement Tells A Story.

Every Visual Supports Emotion.
```

---

# 4. Scene To Shot Architecture

The transformation process:

```
STORY SCENE

↓

SCENE ANALYSIS

↓

SHOT BREAKDOWN

↓

CAMERA DESIGN

↓

MOTION DESIGN

↓

AI VIDEO PROMPT
```

---

# 5. Scene Analysis System

Before creating shots, analyze:

```
Scene Purpose

Character Goal

Emotional State

Important Action

Visual Highlight
```

---

# 6. Shot Definition

A Shot is:

```
A single continuous visual moment

captured from one camera perspective.
```

---

# 7. Shot Components

Every shot requires:

```
Shot ID

Shot Purpose

Camera Type

Subject

Action

Movement

Duration

Emotion
```

---

# 8. Shot Planning Structure

Official format:

```
SHOT

↓

CAMERA

↓

SUBJECT

↓

ACTION

↓

EMOTION

↓

TRANSITION
```

---

# 9. Shot Types

## Establishing Shot

Purpose:

```
Introduce Location

Create Context

Show Environment
```

---

## Wide Shot

Purpose:

```
Show Character Relationship

Show Full Action
```

---

## Medium Shot

Purpose:

```
Show Character Interaction

Balance Emotion And Action
```

---

## Close Up

Purpose:

```
Show Emotion

Highlight Detail
```

---

## Extreme Close Up

Purpose:

```
Create Emotional Focus

Show Important Detail
```

---

# 10. Cinematic Shot Sequence

Recommended sequence:

```
Establishing Shot

↓

Character Introduction

↓

Action Shot

↓

Reaction Shot

↓

Emotional Shot

↓

Resolution Shot
```

---

# 11. Shot Duration System

Duration depends on:

```
Story Rhythm

Audience Attention

Emotional Importance
```

---

Recommended:

```
Fast Action:

1-3 seconds


Normal Story:

3-6 seconds


Emotional Moment:

5-10 seconds
```

---

# 12. Camera Coverage System

Important scenes should include:

```
Opening View

Character View

Action View

Reaction View

Ending View
```

---

# 13. Character Shot Design

Character shots must show:

```
Identity

Emotion

Movement

Personality
```

---

Avoid:

```
Unclear Character

Wrong Scale

Missing Expression
```

---

# 14. Environment Shot Design

Environment shots should communicate:

```
Location

Atmosphere

Story Context

World Identity
```

---

# 15. Action Shot Design

Action shots require:

```
Clear Movement

Clear Goal

Visual Impact

Emotional Response
```

---

# 16. Reaction Shot System

Reaction shots capture:

```
Surprise

Joy

Curiosity

Fear

Achievement
```

---

Purpose:

Create audience connection.

---

# 17. Transition System

Shot transitions include:

```
Cut

Camera Movement

Character Movement

Environmental Transition
```

Mochi learns cooking.

Wide Shot

Shot 3:

Close Up

Shot 4:

Reaction Shot
