# Character OS

Character OS là hệ thống quản lý toàn bộ nhân vật của Wild Story Lab.

## Mục tiêu

- Chuẩn hóa thiết kế nhân vật.
- Đảm bảo tính nhất quán giữa các video.
- Xây dựng Prompt Block có thể tái sử dụng.
- Hỗ trợ Google Flow (Veo 3.1) và các AI video khác.

## Cấu trúc

- Character Standard
- Character Template
- Mochi Character Bible
- Ollie Character Bible
- Prompt Blocks
- QA Checklist
