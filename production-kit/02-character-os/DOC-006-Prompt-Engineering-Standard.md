# Purpose

This document defines the official prompt engineering standard for Wild Story Lab.

Every prompt used for image generation and video generation must follow this structure.

---

# Core Principles

- Character consistency first.
- One clear action per scene.
- One main emotion per scene.
- Cinematic camera movement.
- Child-friendly storytelling.
- Reusable prompt blocks.

---

---

# Character Block

Always begin with the character.

Example:

Mochi, anthropomorphic Shiba Inu, cream golden fur, large emerald green eyes, white hoodie, blue shorts, white sneakers, green scarf.

---

# Action Block

Use only one primary action.

Examples:

- waving
- reading
- cooking
- jumping
- pointing
- dancing

Avoid combining multiple unrelated actions.

---

# Emotion Block

Choose one dominant emotion.

Examples:

- happy
- excited
- curious
- surprised
- proud
- sleepy

---

# Environment Block

Describe only what is visible.

Examples:

- colorful classroom
- cozy kitchen
- sunny park
- magical forest
- space station

---

# Camera Block

Preferred camera shots:

- Close-up
- Medium Shot
- Full Shot
- Tracking Shot
- Orbit Shot
- Dolly In
- Dolly Out

Avoid unnecessary camera changes.

---

# Lighting Block

Recommended lighting:

- Soft morning light
- Warm sunset light
- Bright classroom lighting
- Cozy indoor lighting

Avoid dark or horror lighting.

---

# Animation Style

Always specify:

- Pixar-quality 3D animation
- Disney-inspired style
- cinematic lighting
- highly detailed
- smooth animation
- expressive facial animation

---

# Audio Block

Include:

- Character dialogue
- Ambient sound
- Sound effects
- Background music

---

# Prompt Rules

Always:

- Keep Mochi consistent.
- Keep one clear objective.
- Keep language concise.
- Use positive wording.

Never:

- Change Mochi's identity.
- Mix conflicting emotions.
- Overload a single scene.
- Use violent or frightening descriptions.

# Quality Checklist

Before using a prompt:

- Character matches Character Bible.
- One action only.
- One emotion only.
- Camera is clear.
- Lighting is defined.
- Style is defined.
- Audio is included.
- Child-safe.
