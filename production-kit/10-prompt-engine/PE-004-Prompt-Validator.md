# PE-004 — Prompt Validator

Version: 1.0.0

Status: Production

Owner: Wild Story Lab

Last Updated: 2026-07-23

---

# Purpose

Prompt Validator verifies that every prompt generated by the Prompt Engine complies with Wild Story Lab production standards before it is used for AI video generation.

The validator checks completeness, consistency, compatibility, and quality.

---

# Validation Objectives

- Detect missing information
- Prevent conflicting instructions
- Ensure Character Bible compliance
- Verify Prompt Architecture
- Optimize AI compatibility
- Improve production reliability

✓ Character Block present

✓ Platform Block present

---

# Error Severity

## Critical

Prompt cannot be generated.

Examples

- Missing Story
- Missing Character
- Missing Platform

---

## Warning

Prompt is usable but should be improved.

Examples

- Weak emotion
- Missing ambience
- Generic camera

---

## Suggestion

Optional improvements.

Examples

- Better lighting profile
- Better composition
- Improved transition
