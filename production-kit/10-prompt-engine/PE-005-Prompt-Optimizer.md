# PE-005 — Prompt Optimizer

Version: 1.0.0

Status: Production

Owner: Wild Story Lab

Last Updated: 2026-07-23

---

# Purpose

Prompt Optimizer improves production-ready prompts by increasing clarity, consistency, efficiency, and platform compatibility while preserving the original creative intent.

The optimizer operates after validation and before platform output.

---

# Objectives

- Improve prompt readability
- Remove redundancy
- Increase AI understanding
- Optimize platform compatibility
- Preserve creative intent
- Improve generation consistency

# Optimization Categories

## Structure Optimization

Checks

- Block order
- Logical flow
- Required fields
- Duplicate sections

Actions

- Reorder blocks
- Remove empty blocks
- Merge duplicate content

---

## Language Optimization

Checks

- Ambiguous wording
- Repeated adjectives
- Conflicting instructions
- Overly complex sentences

Actions

- Simplify language
- Replace vague descriptions
- Standardize terminology

---

## Character Optimization

Checks

- Character Bible compliance
- Appearance consistency
- Outfit consistency
- Personality consistency

Actions

- Restore canonical character description
- Remove conflicting traits
- Preserve signature elements

---

## Production Component Optimization

Checks

- Camera
- Motion
- Expression
- Lighting
- Environment
- Audio
- Composition
- Color

Actions

- Replace generic components with standardized profiles
- Remove duplicate instructions
- Improve compatibility

---

## Platform Optimization

Supported

- Google Flow
- Veo 3.1
- Imagen Video
- Runway
- Sora

Actions

- Apply platform-specific wording
- Remove unsupported syntax
- Optimize prompt length
- Adjust formatting

# Optimization Rules

Always

- Preserve story intent.
- Preserve character identity.
- Preserve required blocks.
- Improve clarity.
- Use standardized terminology.

Never

- Change story meaning.
- Invent missing story details.
- Remove required production components.
- Alter Character Bible definitions.
