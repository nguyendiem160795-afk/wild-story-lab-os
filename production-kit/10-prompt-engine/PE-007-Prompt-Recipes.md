# Purpose

Prompt Recipes are reusable production blueprints that combine Story Frameworks, Character OS, Production Components, and Prompt Engine rules into ready-to-generate AI video prompts.

Recipes provide creators with standardized production workflows for common content types.

---

# Recipe Philosophy

A Recipe is not a prompt.

A Recipe is a production formula.

It defines:

- What to use
- When to use it
- How to assemble it

The Prompt Engine transforms a Recipe into a complete production-ready prompt.

---

# Recipe Structure

Every Recipe contains:

- Goal
- Target Audience
- Story Framework
- Character Profile
- Production Components
- Prompt Level
- Platform Profile
- Quality Checklist
- Expected Output

# Quality Checklist

□ Story matches learning objective

□ Character follows Character Bible

□ Environment matches lesson

□ Camera supports clarity

□ Motion is child-friendly

□ Expression matches emotion

□ Lighting is consistent

□ Audio supports learning

□ Platform profile applied

□ Prompt validated

---

# Best Practices

- Start from a Recipe instead of a blank prompt.
- Reuse standardized profiles.
- Keep one learning objective per Recipe.
- Version Recipes independently.
