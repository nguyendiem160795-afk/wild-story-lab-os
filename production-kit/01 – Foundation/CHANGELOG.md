# 1. Purpose (Mục đích)

This document records the official history of changes made to the Foundation module.

Repository-wide changes shall be recorded in `00-master/CHANGELOG.md`.

This changelog includes:

- Foundation documentation
- Foundation architecture
- Engineering principles

- Module baseline revisions

Implementation changes outside the Foundation module are excluded.

# 3. Versioning Policy

The Foundation module follows Semantic Versioning.

```text
MAJOR.MINOR.PATCH
```

| Component | Description |
|-----------|-------------|
| Major | Breaking conceptual or architectural changes |
| Minor | New principles, standards, or documentation |
| Patch | Editorial corrections, clarifications, and minor improvements |
