# Hook System

> Official audience retention hook framework for Wild Story Lab Story Operating System.

---

# 1. Purpose

This document defines the Hook System.

The purpose is to create powerful opening moments that immediately capture audience attention and encourage continued viewing.

The Hook System controls:

- First impression.
- Audience curiosity.
- Early retention.
- Emotional connection.

---

# 2. Hook Definition

A Hook is:

```
The first story moment

designed to capture attention

and create curiosity.
```

---

# 3. Hook Philosophy

Wild Story Lab follows:

```
The Audience Must Feel Something

Before They Understand Everything.
```

---

# 4. Hook Architecture

The Hook System contains:

```
Visual Hook

+

Character Hook

+

Curiosity Hook

+

Emotional Hook

+

Story Question
```

---

# 5. The 0-3 Second Rule

The first three seconds must create:

```
Attention

↓

Interest

↓

Expectation
```

---

Audience reaction:

```
"What is happening?"

"I want to see more."
```

---

# 6. Visual Hook System

## Purpose

Use powerful visuals to stop scrolling.

---

Visual Hook Types:

```
Unexpected Object

Amazing Transformation

Funny Character Action

Large Scale Visual

Bright Color Contrast

Unique Movement
```

---

Examples:

```
Mochi discovers a giant magical ingredient.

A normal object suddenly transforms.

A surprising character reaction appears.
```

---

# 7. Character Hook System

## Purpose

Use character personality to create immediate connection.

---

Character Hooks:

```
Funny Expression

Cute Reaction

Unexpected Behavior

Unique Ability

Emotional Moment
```

---

Rule:

The character must appear recognizable immediately.

---

# 8. Curiosity Hook System

## Purpose

Create a question inside the audience mind.

---

Curiosity Formula:

```
Something Happens

+

Something Is Unknown

=

Want To Know More
```

---

Examples:

```
Why is the kitchen glowing?

What did Mochi discover?

What will happen next?
```

---

# 9. Emotional Hook System

## Purpose

Create immediate emotional connection.

---

Emotional Hook Types:

```
Surprise

Joy

Wonder

Cute Moment

Friendship
```

---

For children:

Preferred emotions:

```
Fun

Magic

Discovery

Happiness
```

---

# 10. Story Question System

Every strong Hook creates a question.

Examples:

```
Can Mochi solve this problem?

What is inside the magical box?

How will the adventure continue?
```

---

The question creates:

```
Curiosity Loop
```

---

# 11. Hook Types Library

## Type 1 - Discovery Hook

Structure:

```
A mysterious thing appears.
```

---

## Type 2 - Transformation Hook

Structure:

```
Something changes unexpectedly.
```

---

## Type 3 - Action Hook

Structure:

```
Something exciting happens immediately.
```

---

## Type 4 - Character Hook

Structure:

```
Character does something memorable.
```

---

## Type 5 - Emotion Hook

Structure:

```
A strong feeling appears instantly.
```

---

# 12. Hook and Story Connection

Hook must connect with:

```
Story Goal

+

Character

+

Main Conflict

+

WOW Potential
```

---

Avoid:

```
Random Visuals

Unrelated Events

False Expectations
```

---

# 13. Hook Escalation System

After the Hook:

```
Attention

↓

Curiosity

↓

Challenge

↓

Discovery

↓

Reward
```

---

A Hook should create momentum.

---

# 14. Short-Form Hook Framework

For 30-60 second videos:

```
0-3s

HOOK


3-10s

QUESTION


10-40s

JOURNEY


40-60s

WOW + REWARD
```

---

# 15. Kids Content Hook Rules

Hooks should be:

```
Simple

Visual

Positive

Easy To Understand

Emotionally Safe
```

---

Avoid:

```
Confusing Mystery

Extreme Fear

Negative Shock
```

---

# 16. Hook Evaluation Checklist

| Requirement | Status |
|---|---|
| Captures attention in 3 seconds | Required |
| Creates curiosity | Required |
| Connects to story | Required |
| Fits character | Required |
| Supports emotion | Required |

---

# 17. Hook Quality Score

Evaluation:

```
Attention

+

Curiosity

+

Emotion

+

Story Connection

=

Hook Strength
```

---

# 18. Canonical Hook Rules

Every story must:

1. Create immediate interest.
2. Introduce curiosity.
3. Connect with the main story.
4. Encourage continuation.
5. Deliver promised value.
