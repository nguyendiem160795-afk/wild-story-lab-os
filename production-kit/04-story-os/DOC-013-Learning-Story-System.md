# Learning Story System

> Official educational storytelling framework for Wild Story Lab Story Operating System.

---

# 1. Purpose

This document defines the Learning Story System.

The purpose is to integrate educational value into stories while maintaining:

- Entertainment quality.
- Emotional connection.
- Character experience.
- Natural learning.

---

# 2. Learning Story Definition

A Learning Story is:

```
A story experience

where characters discover,

practice,

and understand new knowledge

through meaningful events.
```

---

# 3. Learning Philosophy

Wild Story Lab follows:

```
Story First.

Learning Through Experience.

Emotion Creates Memory.
```

---

# 4. Learning Story Architecture

The system contains:

```
Learning Goal

+

Story Experience

+

Character Discovery

+

Interaction

+

Reflection

↓

Learning Outcome
```

---

# 5. Learning Goal System

Every educational story requires:

```
Learning Objective

Knowledge Area

Expected Skill

Behavior Outcome
```

---

Example:

```
Learning Goal:

Learn English vocabulary about animals.
```

---

# 6. Story-Based Learning Model

Learning should happen through:

```
Character Action

↓

Problem

↓

Discovery

↓

Practice

↓

Achievement
```

---

# 7. Character Learning System

Characters are the learning guides.

The character should:

```
Ask Questions

Make Discoveries

Try New Things

Learn From Experience
```

---

Example:

```
Mochi does not know a word.

Mochi discovers the meaning.

Mochi uses the word correctly.
```

---

# 8. Discovery Learning System

Discovery creates natural learning.

Discovery types:

```
New Knowledge

New Skill

New Concept

New Relationship
```

---

# 9. Interactive Learning System

Learning should encourage participation.

Interaction methods:

```
Question

Guessing

Repeating

Choosing

Finding
```

---

Example:

```
Can you find the red apple?
```

---

# 10. Learning Through Challenge

Challenges create opportunities to learn.

Structure:

```
Problem

↓

Attempt

↓

Learning

↓

Success
```

---

# 11. Learning Through Emotion

Emotion strengthens memory.

Learning should connect with:

```
Joy

Curiosity

Achievement

Friendship
```

---

# 12. Educational Content Types

Learning stories may include:

```
Language Learning

Science Discovery

Math Concepts

Life Skills

Creativity

Social Skills
```

---

# 13. Language Learning Framework

For language education:

Include:

```
Vocabulary

Pronunciation

Context

Usage

Repetition
```

---

Example:

```
Apple

↓

Mochi sees an apple.

↓

Mochi says: Apple.
```

---

# 14. Learning Repetition System

Children learn through repetition.

Use:

```
Introduction

Practice

Application

Reward
```

---

# 15. Learning Story Structure

Recommended structure:

```
Hook

↓

Discovery

↓

Learning Challenge

↓

Practice

↓

Achievement

↓

Celebration
```

---

# 16. Learning Scene Requirements

Each learning scene should define:

```
Learning Objective

Character Action

Teaching Moment

Interaction

Reward
```

---

# 17. Age Appropriate Learning Rules

Target:

```
Children 3-8 Years Old
```

Content should be:

```
Simple

Visual

Fun

Interactive

Positive
```

---

Avoid:

```
Complex Explanation

Information Overload

Passive Teaching
```

---

# 18. Learning Story Quality Checklist

| Requirement | Status |
|---|---|
| Learning goal defined | Required |
| Story connection exists | Required |
| Character participates | Required |
| Interaction included | Required |
| Positive outcome achieved | Required |

---

# 19. Learning + Entertainment Balance

The ideal formula:

```
70% Story Experience

+

30% Learning Content
```

---

Learning should never feel separated from the adventure.

---

Story:

Mochi Learns Colors

Learn basic colors.

Mochi discovers red, blue, and yellow.

Friends celebrate learning together.
